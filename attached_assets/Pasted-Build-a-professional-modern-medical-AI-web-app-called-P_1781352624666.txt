API INTEGRATION DETAILS:

This is a Gradio app hosted at: https://hidayatullah099-pneumonia-xray-classifier.hf.space

Use the Gradio REST API (event-based, two-step):

Step 1 - POST the image to start prediction:

POST https://hidayatullah099-pneumonia-xray-classifier.hf.space/gradio_api/call/predict

Content-Type: application/json

Body: { "data": [{"path": "<base64 data URI of the uploaded image>", "meta": {"_type": "gradio.FileData"}}] }

This returns: { "event_id": "<some-id>" }

Step 2 - GET the result using the event_id:

GET https://hidayatullah099-pneumonia-xray-classifier.hf.space/gradio_api/call/predict/<event_id>

This returns a Server-Sent Events (SSE) stream. Parse the "data" event line, which contains a JSON array. The first element is an object:

{

"label": "NORMAL" | "PNEUMONIA",

"confidences": [

{ "label": "NORMAL", "confidence": 0.0-1.0 },

{ "label": "PNEUMONIA", "confidence": 0.0-1.0 }
]

}

Convert the uploaded image file to base64 data URI before sending. Handle the SSE response by reading the stream and parsing lines starting with "data: ".

Handle errors gracefully — if the API call fails or times out, show a friendly error message and suggest the user try again (the free-tier Space may be sleeping and need ~30s to wake up).
